# 👥 User Management System (Backend Simulation)

> A complete user management system built in pure JavaScript that simulates backend operations like adding users, finding users, saving to a database, and handling errors — all in the terminal!

---

## 🎯 Why This Project?

This is my first backend-simulation project built **from scratch** using only JavaScript. No frameworks, no libraries — just pure logic. It helped me understand:

- How to manage data with **arrays and objects**
- How to write clean **functions with parameters**
- How to **validate data** (check for duplicate emails)
- How to **simulate database operations** using Promises
- How to handle errors with **try...catch**
- How to use **modern array methods** (`.find()`, `.forEach()`, `.some()`, `.map()`, `.filter()`)
- How to write **async/await** code like a real backend engineer

---

## ✨ Features

| Feature | Description |
|---------|-------------|
| ➕ **Add User** | Add new users with unique auto-incrementing IDs |
| 🔍 **Find User** | Find a user by their ID |
| 📋 **List All Users** | View all users in the system |
| 🛡️ **Email Validation** | Prevent duplicate emails from being registered |
| 💾 **Database Simulation** | Simulate saving to a database with a 2-second delay |
| ⚠️ **Error Handling** | Gracefully handle database failures without crashing |
| 🔄 **Update User** | Update user information using `.map()` |
| 🗑️ **Delete User** | Remove users using `.filter()` |
| 📊 **User Count** | Get the total number of users |

---

## 🛠️ Technologies Used

- **JavaScript (ES6+)** — Async/Await, Destructuring, Arrow Functions
- **Node.js** — Runtime environment
- **No frameworks** — 100% pure JavaScript logic
